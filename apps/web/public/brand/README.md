# Brand assets

Drop the OPS Tech logo PNGs here. They are referenced from `index.html`,
the sign-in screen, and the top nav.

## Required files

| Filename              | Source                                                 | Used by                              |
| --------------------- | ------------------------------------------------------ | ------------------------------------ |
| `primary-logo.png`    | `Resources/Primary Logo.png` from the OPS brand kit    | Sign-in screen, top nav              |
| `secondary-logo.png`  | `Resources/Secondary Logo.png` from the OPS brand kit  | Email template header, footer        |
| `torch-icon.png`      | `Resources/Torch Icon.png` from the OPS brand kit      | Favicon, small corner marks          |
| `ops-tech-logo-2.png` | `Resources/OPS Tech Logo 2.png` from the OPS brand kit | Optional alternate horizontal lockup |

## Constraints

- Don't recolor, stretch, or crop the Primary or Secondary logos.
- Torch icon may be cropped tight for favicon use.
- See [DESIGN.md](../../../../DESIGN.md) at the repo root for the full brand spec.
